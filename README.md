# MB Brandname

Luxury boutique landing page built with Next.js 14 + Tailwind CSS.

## Features
- Cinematic luxury hero and curated product storytelling
- Responsive layout for desktop and mobile navigation
- Interactive product list with add-to-cart drawer
- Smooth micro-interactions powered by Framer Motion
- Tailwind theme tokens for ivory / gold luxury palette

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production checks
```bash
npm run lint
npm run build
```

## Stack
- Next.js (App Router)
- React 18
- Tailwind CSS
- Framer Motion
- Lucide Icons
